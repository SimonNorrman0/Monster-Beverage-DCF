# Monster Beverage (MNST) - DCF Valuation

An independent discounted cash flow valuation of Monster Beverage Corporation,
built from scratch in Excel.

## Summary
- **Method:** 5-year unlevered FCF projection + Gordon Growth terminal value
- **WACC:** 6.38% (CAPM-derived; β ≈ 0.52, no interest bearing debt)
- **Terminal growth:** 2.5%
- **Implied share price:** ~$59.18 vs. market price ~$97.60
- **Conclusion:** Meaningfully overvalued on intrinsic cash flows -
  consistent with consensus DCF views. The gap reflects brand strength, pricing
  power, and the Coca-Cola distribution moat that a mechanical DCF under-captures.
